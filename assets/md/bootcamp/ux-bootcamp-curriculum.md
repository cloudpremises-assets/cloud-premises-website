# Cloud Premises UX BootCamp – Curriculum

## Program Overview

The **Cloud Premises UX BootCamp** is a structured, cohort-based learning program designed to teach foundational and practical **UX & Product Design** skills to early-career learners. The program emphasizes hands-on learning, collaboration, and mentorship, with **Figma and FigJam** used as core instructional tools throughout the curriculum.

This bootcamp is **education-first** and does not involve client work, agency projects, or commercial services. All projects are fictional or self-initiated for learning purposes only.

---

## Program Details

- **Duration:** 4 weeks  
- **Format:** Remote, cohort-based  
- **Instruction Model:** Instructor-led sessions, guided workshops, and peer collaboration  
- **Tools Used:** Figma, FigJam  
- **Eligibility:** Beginners and early-career learners interested in UX & Product Design  

---

## Learning Outcomes

By the end of the program, students will be able to:

- Apply design thinking principles to solve user problems  
- Conduct basic user research and synthesize insights  
- Create wireframes and high-fidelity UI designs in Figma  
- Build interactive prototypes for usability testing  
- Collaborate effectively using FigJam and Figma  
- Present and critique design work using industry-standard practices  

---

## Weekly Curriculum Breakdown

### **Week 1 – UX Foundations & Design Thinking**

- Introduction to UX and Product Design
- Understanding users and problem framing
- Design thinking frameworks
- Ideation techniques using FigJam
- User personas and user journey mapping

**Activities:**
- FigJam ideation boards  
- Group problem definition workshop  

---

### **Week 2 – Information Architecture & Wireframing**

- Information architecture fundamentals
- User flows and task analysis
- Low-fidelity wireframing
- Accessibility and usability basics
- Intro to Figma interface and components

**Activities:**
- User flow creation in FigJam  
- Wireframes built in Figma  

---

### **Week 3 – UI Design & Prototyping**

- Visual design principles
- Layout, typography, and color systems
- Design systems and reusable components
- High-fidelity UI design in Figma
- Interactive prototyping

**Activities:**
- High-fidelity screens in Figma  
- Clickable prototype creation  

---

### **Week 4 – Testing, Iteration & Presentation**

- Usability testing methods
- Design critique and feedback sessions
- Iteration based on user feedback
- Design handoff fundamentals
- Final project presentation

**Activities:**
- Prototype testing sessions  
- Final design review and presentation  

---

## Assessment & Completion

Students are evaluated based on:

- Participation in workshops and critiques  
- Completion of weekly assignments  
- Final UX project presentation  

A **certificate of completion** is awarded to students who meet program requirements.

---

## Instruction & Mentorship

The bootcamp is led by experienced UX practitioners who provide:

- Live instruction and demos  
- Weekly feedback and critique  
- Mentorship and learning guidance  

---

## Academic Integrity & Usage Policy

- All Figma and FigJam usage is strictly for **educational purposes**
- No client, agency, or revenue-generating work is permitted
- Student work remains part of personal learning portfolios only
